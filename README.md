# 🤖 Prompt Maker - Chrome Extension v2.0

Claude Haiku 4.5를 활용한 초고속 프롬프트 생성 크롬 익스텐션입니다.

## ✨ 주요 기능

- ⚡ **Claude Haiku 4.5**: 2-3초 만에 빠른 분석 (이전 대비 2배 빠름!)
- 📌 **사이드바 인터페이스**: 웹 페이지를 보면서 동시에 사용 가능
- 🔄 **카테고리별 새로고침**: 각 항목마다 새로운 옵션 추가 생성
- 🎯 **키워드 기반 분석**: 단어나 키워드를 입력하면 Claude AI가 자동으로 분석
- 🎭 **6가지 프롬프트 요소**: Persona, Context, Task, Format, Example, Tone
- 🔘 **토글 방식 선택**: 각 요소별로 여러 옵션을 토글 버튼으로 선택
- 📋 **실시간 조합**: 선택한 옵션들이 실시간으로 조합되어 프롬프트 생성
- 📋 **클립보드 복사**: 원클릭으로 생성된 프롬프트를 클립보드에 복사

## 📦 설치 방법

### 1. 파일 준비

이 프로젝트의 모든 파일을 다운로드합니다.

### 2. 아이콘 생성 (선택사항)

아이콘 파일이 없으면 크롬이 기본 아이콘을 사용합니다. 
원하는 아이콘을 사용하려면 다음 크기의 PNG 파일을 준비하세요:
- `icon16.png` (16x16)
- `icon48.png` (48x48)
- `icon128.png` (128x128)

또는 임시로 온라인 아이콘 생성 도구를 사용할 수 있습니다.

### 3. 크롬 익스텐션 로드

1. 크롬 브라우저에서 `chrome://extensions/` 열기
2. 우측 상단의 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. 이 프로젝트 폴더 선택

## 🔑 Claude API 키 설정

1. [Anthropic Console](https://console.anthropic.com/)에서 API 키 발급
2. 익스텐션 팝업에서 API 키 입력 후 "저장" 버튼 클릭

## 🚀 사용 방법

### 1단계: 사이드바 열기
```
크롬 툴바의 Prompt Maker 아이콘 클릭
→ 우측에 사이드바 자동 오픈
```

### 2단계: 키워드 입력
```
예: "블로그 포스트, SEO 최적화, 친근한 톤"
예: "마케팅 이메일, B2B, 전문적"
```

### 3단계: 분석하기
"✨ 분석하기" 버튼 클릭 → 2-3초 대기 (Haiku 4.5는 매우 빠릅니다!)

### 4단계: 옵션 선택
6가지 카테고리별로 제안된 옵션들 중 원하는 것을 클릭:
- 🎭 **Persona**: 역할이나 전문성
- 📝 **Context**: 상황이나 배경
- ✅ **Task**: 구체적인 작업
- 📋 **Format**: 출력 형식
- 💡 **Example**: 참고 예시나 스타일
- 🎵 **Tone**: 어조나 분위기

### 5단계: 🔄 새로고침 (NEW!)
각 카테고리 옆의 **🔄 버튼**을 클릭하면 새로운 옵션 3-5개가 추가됩니다!
- 기존 옵션 유지 + 새 옵션 추가
- 마음에 드는 옵션 찾을 때까지 반복 가능
- 카테고리별로 독립적으로 작동

### 6단계: 프롬프트 확인 및 복사
선택한 옵션들이 자동으로 조합되어 하단에 표시됩니다.
"📋 클립보드에 복사" 버튼으로 생성된 프롬프트를 복사할 수 있습니다.

## 🛠️ 기술 스택

- **Manifest V3**: 최신 크롬 익스텐션 표준 (Side Panel API 포함)
- **Claude Haiku 4.5**: Anthropic의 최신 고속 모델 (2025년 10월)
- **Vanilla JavaScript**: 프레임워크 없이 순수 JS로 구현
- **Modern CSS**: 사이드바 최적화 UI, 그라데이션과 애니메이션
