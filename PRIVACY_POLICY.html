🔒 PCTFET 프롬프트 메이커 개인정보처리방침

최종 업데이트: 2025년 1월
버전: 3.0.0

PCTFET 프롬프트 메이커("본 확장 프로그램")는 사용자의 개인정보 보호를 매우 중요하게 생각합니다. 본 개인정보처리방침은 본 확장 프로그램이 수집, 사용, 저장하는 데이터에 대해 명확하게 설명합니다.

1. 수집하는 데이터

필수 데이터 (서비스 제공을 위해 필요)

데이터 유형 저장 위치 사용 목적
API 키 로컬 스토리지
(사용자 기기 내부)		 AI 서비스 API 인증 및 프롬프트 생성 기능 제공
사용자 입력 텍스트 메모리
(세션 종료 시 삭제)		 AI 기반 프롬프트 생성 기능 제공
설정 정보 로컬 스토리지 사용자 선호 설정 저장 (AI 제공자, 모델 선택)

⚠️ 전송 정보:

  • API 키: 선택한 AI 서비스 제공자의 API 서버로 전송 (인증 목적)
  • 사용자 입력 텍스트: 선택한 AI 서비스 제공자의 API 서버로 전송 (프롬프트 생성 요청)

2. 수집하지 않는 데이터

✓ 수집하지 않음
본 확장 프로그램은 다음 데이터를 수집하지 않습니다.

  • 개인 식별 정보 (이름, 이메일, 전화번호 등)
  • 브라우징 기록
  • 웹사이트 방문 정보
  • 위치 정보
  • 기기 정보
  • 사용 패턴 분석 데이터

3. 데이터 사용 목적

수집된 데이터는 오직 다음 목적으로만 사용됩니다:

  1. 프롬프트 생성 기능 제공: 사용자가 입력한 키워드를 기반으로 AI가 프롬프트 옵션을 생성
  2. API 인증: 선택한 AI 서비스 제공자의 API에 접근하여 서비스 이용
  3. 사용자 설정 저장: 사용자의 AI 제공자 및 모델 선택을 저장하여 편의성 제공

4. 데이터 저장 및 보안

데이터 저장 위치:

  • 로컬 스토리지: 모든 데이터는 사용자의 기기 내부 Chrome 확장 프로그램 로컬 스토리지에만 저장됩니다.
  • 외부 서버: 본 확장 프로그램은 자체 서버를 운영하지 않으며, 사용자 데이터를 외부 서버에 저장하지 않습니다.

데이터 전송:

  • 사용자가 선택한 AI 서비스 제공자(Anthropic, OpenAI, Google Gemini, Ollama)의 API 서버로만 데이터가 전송됩니다.
  • 전송되는 데이터: API 키(인증용), 사용자 입력 텍스트(프롬프트 생성 요청)
  • 전송 방식: HTTPS 암호화 통신

데이터 보안:

  • API 키는 로컬 스토리지에 저장되며, AI API 인증 시에만 전송됩니다.
  • 사용자 입력 텍스트는 세션 동안에만 메모리에 저장되며, 확장 프로그램을 닫으면 자동으로 삭제됩니다.

📌 참고: 외부 서버에 데이터를 저장하지 않습니다. 모든 처리는 브라우저 내에서 이루어지며, AI API 호출 시에만 선택한 서비스로 데이터가 전송됩니다.

5. 데이터 공유 및 제3자 제공

✓ 공유하지 않음
본 확장 프로그램은 사용자 데이터를 제3자에게 판매, 거래, 공유하지 않습니다.

AI API 서비스 관련:

  • 사용자가 직접 선택한 AI 서비스로만 키워드 전송
  • 사용자가 직접 제공한 API 키로 인증
  • 각 AI 서비스의 개인정보처리방침은 해당 서비스 제공자가 관리

AI 서비스별 개인정보처리방침:
• Anthropic (Claude): anthropic.com/privacy
• OpenAI: openai.com/policies/privacy-policy
• Google Gemini: policies.google.com/privacy
• Ollama: 로컬 실행 시 데이터 전송 없음

6. 사용자 권리 및 선택

데이터 수집 동의:

본 확장 프로그램을 설치하고 사용함으로써, 사용자는 본 개인정보처리방침에 명시된 데이터 수집 및 사용에 동의하는 것으로 간주됩니다. Chrome 웹 스토어 등록정보에서 본 개인정보처리방침을 확인할 수 있습니다.

데이터 삭제:

사용자는 언제든지 다음 방법으로 저장된 데이터를 삭제할 수 있습니다:

  1. API 키 삭제: 확장 프로그램 설정에서 API 키를 삭제하거나 변경
  2. 모든 데이터 삭제: Chrome 설정 > 확장 프로그램 > PCTFET 프롬프트 메이커 > 데이터 삭제
  3. 확장 프로그램 제거: 확장 프로그램을 제거하면 모든 로컬 데이터가 자동으로 삭제됩니다.

오프라인 모드 (Ollama 사용):

💡 오프라인 모드: 사용자는 Ollama를 선택하여 로컬에서 AI 모델을 실행할 수 있습니다. Ollama 모드를 사용하면 사용자 입력 텍스트가 인터넷으로 전송되지 않으며, 모든 처리가 사용자의 기기에서 이루어집니다.

데이터 유형 보관 기간
